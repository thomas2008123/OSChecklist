## 1. Visual / Interface Designs

 - [ ] For each design, create a complete user interface

 - [ ] Include a clear demonstration of the user journey

 - [ ] Provide justification for the colour schemes used

 - [ ] Follow W3C accessibility guidelines (contrast, font size, ARIA, etc.)

 - [ ] State the front-end requirements

 - [ ] State the back-end requirements

 - [ ] Evaluate the security of the system (explain who manages security, not how it’s implemented)
